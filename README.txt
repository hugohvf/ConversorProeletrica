PRO-Elétrica é um software para projetos elétricos residencias e prediais.
Trabalha em conjunto com o autocad e se baseia na NBR ISO 8995-1 para o cálculo luminotécnico da área usando o método de Lumens
O software gera um txt com todos os cálculos feitos para chegar ao fluxo luminoso desejado. Cita quantas lâmpadas devem ser usadas em cada espaço e o tipo escolhido
O conversor nada mais faz que gerar um csv com todas essas informações para gerar uma planilha no excell com os cálculos

COMO USAR:
Para usar basta abrir o executavel com o txt gerado pelo PRO-Elétrica no mesmo diretório que o executável.
O txt tem que estar nomeado como "Dados.txt"
